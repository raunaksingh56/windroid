## вңЁ What is WinDroid?

**WinDroid** is an open-source Android app that turns your phone into a bootable Windows USB creator. Plug in a USB OTG drive, pick your ISO, and flash вҖ” the USB boots on any PC automatically.

**Supports every modern Windows version:**

| Version | Answer File | Notes |
|---------|-------------|-------|
| рҹӘҹ Windows XP | `winnt.sif` | Legacy text-format, full unattend |
| рҹӘҹ Windows 7 | `autounattend.xml` | ei.cfg injection, activation skip |
| рҹӘҹ Windows 8 / 8.1 | `autounattend.xml` | Online account bypass, SmartScreen off |
| рҹӘҹ Windows 10 | `autounattend.xml` | Telemetry & Cortana disable |
| рҹӘҹ Windows 11 | `autounattend.xml` | Full TPM / SecureBoot / RAM bypass |

## рҹҺҜ How to Use

```
Step 1 в”Җв”Җв–ә Step 2 в”Җв”Җв–ә Step 3 в”Җв”Җв–ә Step 4 в”Җв”Җв–ә Step 5
 Get ISO    Plug USB   Open App   Configure   Flash!
```

1. **Get a Windows ISO** вҖ” download from Microsoft or use an existing one
2. **Plug in USB OTG drive** вҖ” app opens automatically
3. Tap **Select ISO** вҶ’ pick your Windows ISO file
4. Tap **Select USB** вҶ’ pick the OTG drive
5. Tap **Windows Version** вҶ’ pick XP / 7 / 8 / 10 / 11 (or leave on Auto-Detect)
6. Configure tweaks (optional вҖ” defaults work great)
7. Tap **Flash Windows to USB**
8. Plug USB into your PC вҶ’ boot вҶ’ Windows installs automatically!

## рҹ”§ How It Works

### ISO 9660 Parser

```
ISO file
  в””в”Җв”Җ Sector 16 в”Җв–ә Primary Volume Descriptor
        в””в”Җв”Җ Root directory record
              в”ңв”Җв”Җ /sources/install.wim  (or install.esd)
              в”ңв”Җв”Җ /EFI/BOOT/bootx64.efi
              в”ңв”Җв”Җ /bootmgr
              в””в”Җв”Җ ... (all files extracted)
```

WinDroid implements ISO 9660 parsing entirely in Kotlin:
- Reads the **Primary Volume Descriptor** at sector 16
- Detects **Joliet extensions** (sector 17+) for Unicode long filenames
- Walks the directory tree recursively
- Streams each file directly to the USB drive

### install.wim Splitting

Windows 11 ISOs have `install.wim` or `install.esd` > 4 GB. FAT32 has a 4 GB file limit. WinDroid automatically splits these into `.swm` chunks that Windows Setup handles natively:

```
install.wim (5.2 GB)
  вҶ’ install.swm   (3.8 GB)  вҶҗ chunk 1
  вҶ’ install2.swm  (1.4 GB)  вҶҗ chunk 2
```

### Answer File Generation

Different Windows versions need different answer files:

```
Windows XP  вҶ’  winnt.sif       (INI format, placed in ISO root)
Windows 7+  вҶ’  autounattend.xml (XML schema, placed in USB root)
Windows 7+  вҶ’  sources/ei.cfg   (optional edition override)
```

WinDroid generates all required files automatically based on your version selection.

### Dual Boot (UEFI + BIOS)

```
USB Drive
в”ңв”Җв”Җ EFI/BOOT/bootx64.efi   вҶҗ UEFI 64-bit (all modern PCs, 2012+)
в”ңв”Җв”Җ EFI/BOOT/bootia32.efi  вҶҗ UEFI 32-bit (Bay Trail tablets вҖ” root only)
в”ңв”Җв”Җ bootmgr                вҶҗ Legacy BIOS fallback
в””в”Җв”Җ Boot/BCD               вҶҗ Boot Configuration Database
```
